# Dark Souls Fandom ⚔️🔥
> "Se você perder sua determinação, ficará Vazio." — Solaire de Astora.

## 📖 Sobre o Projeto

O **Dark Souls Fandom** é uma aplicação web idealizada para conectar a comunidade de Dark Souls I. Inspirado na filosofia de cooperação mútua apresentada pelo criador do jogo, Hidetaka Miyazaki, o projeto visa criar um ambiente acolhedor onde jogadores podem trocar conhecimentos, pedir ajuda e discutir a rica história de Lordran.

O objetivo principal é mitigar a taxa de desistência de novos jogadores, oferecendo o suporte necessário para que a dificuldade punitiva do jogo se torne um desafio prazeroso e recompensador, eliminando sentimentos de frustração.

## 🌟 Contexto e Inspiração

O projeto nasceu de uma paixão pessoal iniciada em 2023, após o contato com a obra-prima da FromSoftware. A jornada através de áreas desafiadoras e encontros marcantes — especialmente com o icônico **Solaire de Astora** — revelou a beleza do sistema cooperativo.

Miyazaki concebeu o modo multiplayer baseado em uma experiência real de ser ajudado por estranhos em uma estrada com neve, onde as pessoas se ajudam sem saber quem são e sem esperar nada em troca. É esse espírito de "cooperação altruísta" que o Dark Souls Fandom busca replicar.

## 🚀 Funcionalidades

A aplicação está dividida em pilares principais para melhor atender o usuário:

- **Dashboard Personalizada:** Visualização de dados sobre os assuntos do momento e as interações do próprio usuário através de gráficos e KPIs.
- **Fórum Comunitário:** Espaço dividido em quatro categorias essenciais:
    - 🆘 **Ajuda:** Para jogadores travados em chefes ou áreas específicas.
    - 💡 **Dicas:** Compartilhamento de segredos, itens e mecânicas.
    - 📜 **História (Lore):** Discussões profundas sobre o universo do jogo.
    - ⚔️ **Builds:** Recomendação de atributos e equipamentos.
- **Site Institucional:** Informações detalhadas sobre o projeto e a história de Dark Souls I para novos aventureiros.
- **Sistema de Autenticação:** Login e cadastro seguro para garantir a identidade única de cada guerreiro na comunidade.

## 🛠️ Requisitos do Sistema

O projeto contempla os seguintes requisitos técnicos e funcionais:

- [x] **Interface Web:** Design focado na experiência do usuário.
- [x] **Backend & Segurança:** Sistema de cadastro que impede duplicidade de e-mails ou nomes de usuário.
- [x] **Banco de Dados:** Armazenamento em ambiente virtual (Cloud/VM).
- [x] **Análise de Dados:** Transformação de dados de discussão em visualizações gráficas (Dashboards).

## 📊 Estrutura da Dashboard

O sistema gera indicadores para rápido entendimento:
1. **Assunto da Comunidade:** Qual tópico está em alta no momento.
2. **Engajamento Pessoal:** Qual área o usuário mais frequenta e contribui.

## 📦 Como rodar o projeto

Siga os passos abaixo para executar o projeto na sua máquina:

### 1. Clone o repositório

```bash
git clone https://github.com/gustavoLimaAraujo/dark-souls-fandom.git
```

### 2. Acesse a pasta do projeto

```bash
cd dark-souls-fandom
```

### 3. Instale as dependências

```bash
npm install
```

ou simplesmente:

```bash
npm i
```

### 4. Execute o projeto

```bash
npm start
```

ou, dependendo da configuração:

```bash
npm run dev
```
